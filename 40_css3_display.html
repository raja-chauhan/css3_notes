CSS3 display

Concept : giving one tag power to other tag to behave differently as compare to normal
  • None -> use this to hide the and leave area
  • Inline -> to display everthing in one line
  • block -> to display everythin in individual line
  • Inline-block -> display in one line but span and margin will apply like in block
  • list-item -> display as a list items
