CSS Basic Selector

Types of basic selectors

  1. By tag name -> multiple time use
  2. Class Name -> multiple time use
  3. Id name -> one time use
Example :
tag name -> h1 { css properties }
class name -> .className {css properties}
ID name -> #id {css properties}
