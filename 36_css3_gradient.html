CSS3 gradient

Use this property if you want to use more than one color in you background you can use multiple color at once

Diffrent gradient type

  1. Linear gradient -> run from one side to other(top-bottom left-right right-left bottom-top)
  2. radical gradient -> run from point (center right-bottom left-bottom left-top)
  3. repeated linear gradient
  4. repeated radical gradient
