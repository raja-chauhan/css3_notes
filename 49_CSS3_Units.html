CSS3 Measurment Units

  1. px(pixel)
  2. %(percentage)
  3. em used for font = 16px depend on parent
  4. rem -> used for font (work according to root/browser)
  5. vw(viewport width)
  6. vh(viewport height)
  7. vmax (viewport maximum) -> what ever max in width and height that will be used
  8. vmin (viewport minimum) -> what ever max in width and height that will be used
