CSS3 position

  1. static
  2. relative -> use left right top bottom along with it to work (current position)
  3. absolute -> z-index increase and start display on other app, use left right top bottom along with it to work (screen position)
  4. fixed -> display along with scroll z-index increase
  5. sticky -> display on scroll up or down
